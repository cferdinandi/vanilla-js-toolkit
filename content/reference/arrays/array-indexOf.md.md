Get the index of the first matching item in an array. Returns `-1` if no matching item is found.

```javascript
var sandwiches = ['turkey', 'tuna', 'blt', 'pb&jb'];

// returns 1
sandwiches.indexOf('tuna');

// returns -1
sandwiches.indexOf('ham');
```
